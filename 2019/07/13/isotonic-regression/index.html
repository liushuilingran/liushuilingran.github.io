  大部分时候，我们会在括号前加上权重w_i。解决这个问题的一个方法就是 pool adjacent violators algorithm(PAVA) 算法。粗略的讲，PAVA算法的过程描述如下：

  我们从左边的y_1开始，右移y_1直到我们遇到第一个违例(violation)即y_i < y_i+1，然后，我们用违例之前的所有y的平方替换这些y，以满足单调性。我们继续这个过程，直到我们最后到达y_n。
